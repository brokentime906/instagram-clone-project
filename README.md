- [ ] Like / Unlike a photo

- [ ] Search by Location

- [ ] See the full photo
- [ ] Edit my profile
- [ ] Create account
- [ ] Log in
- [ ] Upload a photo
- [ ] Edit the photo(Delete Photo)
- [ ] See the feed
